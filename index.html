Avaliação Diagnóstica

CSS é a linguagem que usamos para estilizar um documento HTML. O CSS descreve como os elementos HTML devem ser exibidos, usando como base classes linkadas em diferentes elementos do HTML para defini-lo. O CSS usa além da declaração da classe, chaves para definir o que estará contido no estilo daquele elemento. Podem ser utilizadas classes, ID's ou a tag HTML do elemento em si.

Link para o site da imagem.

Usos do HTML:

  • Desenvolvimento web. Os desenvolvedores usam códigos HTML para projetar como um navegador vai exibir os elementos das páginas, como textos, hiperlinks e arquivos de mídia.
  • Navegação na internet. Os usuários podem navegar facilmente e inserir links entre páginas e sites relacionados, já que o HTML é amplamente usado para incorporar hiperlinks.
  • Documentação. O HTML torna possível a organização e a formatação de documentos, de maneira similar ao Microsoft Word.

Formulário

Box Model

Box model é um conceito fundamental no desenvolvimento da Web que determina como os elementos em uma página da Web são dimensionados e posicionados, de forma similar a conteiners que definem quanto da página o elemento vai abranger.

Exemplos:

  • Margem
  • Border
  • Padding
  • Content

Pseudo Classes

Uma pseudo classe é usada para definir um estado específico para algum elemento dependendo do seu estado.

Exemplos:

  • :hover
  • ::after
  • ::before
